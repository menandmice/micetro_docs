Micetro user interface(s)
-------------------------

Micetro currently uses two user interfaces: the new, browser-based :ref:`web-application` and the older, Windows-only :ref:`console`.

.. note::
  Migrating features from the Management Console to the Web Application is ongoing, and new features are not added to the Console.
