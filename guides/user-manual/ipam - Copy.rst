IPAM
=====

.. note::

  * To access IP Address Management (IPAM) in Micetro, you must enter the license key for the IPAM module.

  * This information applies to the Micetro Web Interface. For information about IP address management in the Management Console, see :ref:`console-ipam`.

IP Address Management (IPAM) is where you manage all aspects of your network infrastructure. This section explains how to handle IP addresses within your address spaces. You'll learn how to configure networks, view, monitor, and edit the properties of individual IP addresses, as well as how to integrate with Active Directory (AD) sites. Additionally, it covers managing DHCP superscopes and registering and monitoring devices.
